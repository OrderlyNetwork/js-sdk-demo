# Orderly SDK demo

## Getting started

1. Clone the code:

```
git clone git@github.com:OrderlyNetwork/js-sdk-demo.git
cd js-sdk-demo
```

2. Install the dependencies

```
npm install
```

3. Run

```
npm start
```
